## 지원하는 기능

- `@Controller()`
  - `@Get()`
  - `@Post()`
  - `@Delete()`
  - `@Put()`
  - `@Patch()` 
- `@Component()`
- `@Autowired()`

## 설치
`npm i @miensoap/express-annotations`
